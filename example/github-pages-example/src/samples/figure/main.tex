\section{Including Images}
BusyTeX supports including images directly in your documents.
Figure~\ref{fig:logo} shows the TeXlyre logo included as a PNG file.

\begin{figure}[H]
  \centering
  \includegraphics[width=0.4\textwidth]{TeXlyre.png}
  \caption{The TeXlyre logo}
  \label{fig:logo}
\end{figure}

\section{Scaling}
Images can be scaled to any size using the \texttt{width} or \texttt{height} options.
